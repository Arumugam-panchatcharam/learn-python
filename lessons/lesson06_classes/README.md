# Lesson 06 — Classes

## Learning Goals

- Model domain concepts with classes
- Choose composition over inheritance when appropriate
- Use common magic methods intentionally

## Important Notes

- Read `theory.py` first for focused examples.
- Study `examples.py` from simple to professional.
- Attempt `exercises.py` before opening `solutions.py`.
- Keep code out of this README — notes live in [notes/lesson06.md](../../notes/lesson06.md).

## Pitfalls

- Deep inheritance hierarchies
- Putting too much logic in `__init__`
- Overusing magic methods without clear intent
